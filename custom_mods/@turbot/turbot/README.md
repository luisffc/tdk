# Turbot

The `@turbot/turbot` mod provides base definitions and resources for the entire
Turbot application. It is used as a dependency by almost every mod.

* Fundamental resource types, e.g. Folder.
* Common categories, e.g. Cloud, Storage, Compute.
* Turbot internal resource types, e.g. Mod, Control Type.

See the [Turbot Concepts documentation](https://turbot.com/v5/docs/concepts)
for an overview of the fundamentals.
